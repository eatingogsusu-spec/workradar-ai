# PRD: TeamMemory

## Goal
TeamMemory는 팀 프로젝트의 문서, Todo, 이슈, AI 질의응답, 주간 보고, 인수인계 정보를 프로젝트 단위로 모아 후임자가 빠르게 맥락을 파악하도록 돕는 인수인계 서비스다.

## Users
- 프로젝트를 진행하는 5명 내외의 팀원
- 프로젝트를 넘겨받는 신규 담당자 또는 후임자
- Todo, 이슈, 회의록, 자료 업로드 상태를 한 화면에서 확인해야 하는 팀 리더

## Core Flows
1. 팀원이 프로젝트에 참여한다.
2. 팀원이 문서, 회의록, API 명세 같은 자료를 업로드한다.
3. 시스템이 문서를 chunk로 나누고 외부 vector store에 색인한다.
4. AI가 문서에서 Todo, 이슈, 요약, 인수인계 누락 항목을 추출한다.
5. 사용자는 대시보드에서 진행률, 이번 주 Todo, 미해결 이슈, 인수인계 점수를 확인한다.
6. 사용자는 AI 챗봇에 질문하고 문서/chunk 출처가 연결된 답변을 받는다.

## MVP Features
- 프로젝트 및 프로젝트 멤버 관리
- 자료 업로드 메타데이터 저장
- 문서 chunk 및 외부 vector store 참조 저장
- 프로젝트별 Todo, 이슈, 채팅 메시지 저장
- 주간 보고서와 인수인계 보고서 저장
- 대시보드 집계 쿼리 제공

## Out Of Scope
- 결제, 조직별 요금제, 권한 고도화
- 실제 파일 저장소 구현
- 실제 embedding 생성 및 vector search 구현
- 실시간 협업 편집
- 이메일/Slack/Notion 실연동

## Success Criteria
- 모든 핵심 업무 데이터가 `project_id`로 분리된다.
- 팀원 5명 샘플 데이터로 대시보드 화면을 구성할 수 있다.
- Todo, 이슈, 문서, 챗봇, 인수인계 화면에 필요한 조회 쿼리가 제공된다.
- DB 담당자가 백엔드/프론트 담당자에게 공유할 수 있는 DDL과 테이블 정의서가 준비된다.
